    General Description

    This service aims to provide a website that allows users to easily and effectively manage various user-created books.

    To access all functionalities, it is recommended that users create an account or log in with an existing one.

    The following features are accessible through this service:

    Feature Functionality Requires Registration
    Display Books Users can view and search for books created by others. Users can also view each books' details including ratings, comments, and option to purchase. No
    Post Book Users can create and post a book for others to access. No
    My Books Users can view their own published books and delete if desired. Yes
    Rate Users can rate a book from 1-5. Each book's average ratings will be displayed. Yes
    Comment Users can leave comments on a book. Each comment displays the user, message, and date of creation. Yes
    Shopping Cart Users can add books to their shopping cart to purchase. Yes
    Message Users can message others. Yes
