    General Description

    This service aims to provide a website that allows users to easily and effectively manage various user-created books.

    To use this service, users must create an account or log in with an existing one.

    The following features are accessible through this service:

    Feature Functionality
    Display Books Users can view and search for books created by others. Users can also view each books' details and leave ratings/comments.
    Post Book Users can create and post a book for others to access.
    My Books Users can view their own published books and delete if desired.
    Shopping Cart Users can add books to their shopping cart to purchase.
